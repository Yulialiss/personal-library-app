Ідея проєкту
Проєкт представляє собою застосунок Бібліотека книг, який призначений для зручного зберігання та впорядкування особистої колекції книг.

Користувач може:
-додавати нові книги до своєї бібліотеки
-переглядати повний список книг
-шукати книги за назвою або автором
-відстежувати статус читання кожної книги

Функціонал
-Додавання нових книг
-Редагування інформації про книгу
-Видалення книг
-Пошук за назвою або автором
-Перегляд повного списку книг
-Відмітка статусу книги : прочитано,читаю зараз,планую прочитати
-Збереження даних у базі даних

-Засоби розробки
Python - Tkinter (графічний інтерфейс)

Інтерфейс 
-блідо-пастельні кольори як основна палітра
-яскраві матові акценти для кнопок та елементів керування
